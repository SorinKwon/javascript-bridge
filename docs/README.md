# 다리 건너기

## 🔖 프로젝트 소개

### 우테코 프리코스 4주차 다리 건너기 미션입니다.
### 
---

## 📋 기능 목록

### 입력
* 다리 길이 입력
* 이동할 칸 입력 (U or D)
* 게임 재시작/종료 여부 입력 (R or Q)

### 출력
* 게임 시작 안내문 출력
* 다리 길이 입력 안내문 출력
* 칸 선택 안내문 출력
* 다리 출력
* 이동 가능/불가능 칸 출력
* 예외 시 에러 문구 출력

### 다리 생성
* 입력받은 다리 길이로 다리 생성


###
* 

---

## 📛 예외 목록

### 입력
* 다리 길이는 3 이상 20 이하의 숫자를 입력
* 이동할 칸은 U와 D중 하나의 문자를 입력
* 게임 재시작/종료 여부는 R과 Q 중 하나의 문자를 입력
